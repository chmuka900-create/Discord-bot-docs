# Polityka prywatności

## 1. Informacje ogólne
Ta polityka prywatności opisuje, jakie dane może przetwarzać nasz bot Discord oraz w jaki sposób są one wykorzystywane.

## 2. Zbierane dane
Bot może przechowywać podstawowe informacje potrzebne do działania, takie jak:
- ID użytkownika Discord,
- ID serwera Discord,
- ustawienia konfiguracji bota.

## 3. Wykorzystanie danych
Dane są wykorzystywane wyłącznie do działania funkcji bota oraz poprawy jego działania.

## 4. Udostępnianie danych
Nie sprzedajemy ani nie udostępniamy danych użytkowników osobom trzecim.

## 5. Bezpieczeństwo
Staramy się chronić przechowywane dane przed nieautoryzowanym dostępem.

## 6. Usunięcie danych
Użytkownik może poprosić administrację o usunięcie swoich danych, jeśli bot takie dane przechowuje.

## 7. Zmiany polityki
Polityka prywatności może zostać zmieniona w dowolnym momencie.
