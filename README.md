# Discord-bot-docs
# Warunki korzystania z usługi

## 1. Postanowienia ogólne
Korzystając z naszego bota Discord, akceptujesz poniższe warunki korzystania z usługi.

## 2. Korzystanie z bota
Bot jest przeznaczony do użytku zgodnego z zasadami Discord oraz obowiązującym prawem. Zabrania się wykorzystywania bota do działań szkodliwych, spamowania, oszustw lub łamania regulaminu.

## 3. Dane użytkowników
Bot może przechowywać podstawowe dane wymagane do działania funkcji bota, takie jak identyfikator użytkownika Discord, identyfikator serwera oraz ustawienia bota.

## 4. Dostępność usługi
Nie gwarantujemy, że bot będzie działał bez przerw. Możemy aktualizować, zmieniać lub zakończyć działanie wybranych funkcji.

## 5. Odpowiedzialność
Nie ponosimy odpowiedzialności za szkody wynikające z nieprawidłowego użycia bota.

## 6. Zmiany warunków
Warunki korzystania mogą zostać zmienione w dowolnym momencie. Aktualna wersja będzie dostępna na tej stronie.

## 7. Kontakt
W przypadku pytań skontaktuj się z administracją serwera Discord.
